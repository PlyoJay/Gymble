# Gymble

헬스장 회원, 상품, 출석, 결제 관리를 위한 WPF 기반 ERP MVP입니다.

## 기술 스택
- C# / .NET 8 WPF
- MVVM
- SQLite
- Dapper
- MaterialDesignInXaml
- CommunityToolkit.Mvvm

## 주요 기능
- 회원 등록/수정/삭제
- 회원 검색 및 페이징
- 상품/이용권 관리
- 상품 구매
- 출석 관리
- 대시보드

## 실행 방법
1. Visual Studio 2022 실행
2. Gymble.sln 열기
3. NuGet Restore
4. Gymble 프로젝트 실행
